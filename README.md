# Seppo.js

Usage:
```
npm install "apvilkko/seppojs"
```

In your HTML:
```
...
<head>
  ...
  <link rel="stylesheet" href="path/to/seppojs/dist/seppo.css">
  ...
</head>
<body>
  ...
  <script src="path/to/seppojs/dist/seppo.js"></script>
  <script>SeppoJS.activate();</script>
</body>
```
